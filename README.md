# e-commercial

1. Установите зависимости:

```
npm install
```

2. Запуск в режиме разработки:

```
npm run dev
```

3. Сборка для продакшна:

```
npm run build
```

4. Предпросмотр собранной версии:

```
npm run preview
```

**Лицензия**

Проект — учебный/портфолио. Используйте по своему усмотрению.
